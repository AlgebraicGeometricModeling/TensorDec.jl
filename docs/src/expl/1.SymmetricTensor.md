
We consider symmetric tensors or equivalently homogeneous polynomials, in the following variables:


```julia
using TensorDec, DynamicPolynomials
X = @polyvar x0 x1 x2;
```

```julia
d=4; F = (x0+x1+0.75x2)^d + 1.5*(x0-x1)^d -2.0*(x0-x2)^d
```
